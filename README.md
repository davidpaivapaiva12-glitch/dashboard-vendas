# 📊 Dashboard de Vendas

Projeto de análise de dados com Python, utilizando Streamlit e Plotly.

## 🚀 Funcionalidades
- Faturamento total
- Vendas por produto
- Vendas por mês
- Dashboard interativo

## 🛠️ Tecnologias
- Python
- Pandas
- Streamlit
- Plotly

## ▶️ Como executar
```bash
pip install -r requirements.txt
streamlit run app.py
